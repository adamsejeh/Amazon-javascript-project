# Amazon Clone Project

## Description
This is an Amazon checkout clone built with JavaScript, HTML, and CSS.

## Features
- Add to cart
- Remove products
- Update delivery options
- Payment summary
- LocalStorage support

## Technologies Used
- JavaScript
- HTML
- CSS
